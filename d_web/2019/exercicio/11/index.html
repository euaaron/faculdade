FAÇA COMO NO VÍDEO:

Resposta:

Você me ama?

Desenvolvimento:

Para resolver esse exercício, visando impedir que alguma mudança ocorresse na página ao passar o cursor sobre o botão "Sim", foram utilizados dois eventos onmouseover e duas chamadas de diferentes funções, sendo uma para cada evento. Tais eventos foram adicionados aos botões, os quais identifiquei como btn1 e btn2, enquanto as funções desses eventos nomeei como botaoUm() e botaoDois(). Veja abaixo o código na íntegra:

HTML

<p>Você me ama?</p> <button id="btn1" onmouseover="botaoUm()">Sim</button> <button id="btn2" onmouseover="botaoDois()">Não</button>

JavaScript

//Um contador para identificar oquê acontecerá ao passar o cursor sobre o botão "Não" estado = 0; function botaoUm() { um = document.getElementsByTagName("button")[0].innerHTML; dois = document.getElementsByTagName("button")[1].innerHTML; //Verifica se é o "Não", caso seja, executa uma função que troca o estado do botão if (um == "Não") { troca(); } } function botaoDois() { um = document.getElementsByTagName("button")[0].innerHTML; dois = document.getElementsByTagName("button")[1].innerHTML; //Verifica se é o "Não", caso seja, executa uma função que troca o estado do botão if (dois == "Não") { troca(); } } function troca() { switch(estado) { case 2: //Volta para os valores originais, incluindo o do estado. btn2.style.top = "0px"; btn1.innerHTML = "Sim"; btn2.innerHTML = "Não"; estado = 0; break; case 1: //Troca os valores dos botões e muda o estado para 2 btn1.innerHTML = "Não"; btn2.innerHTML = "Sim"; estado++; break; default: //Move o botão para baixo e muda o estado para 1 btn2.style.top = "128px"; estado++; break; } }
