Olá mundo!

Como todo exercício inicial, começo este com a clássica frase mundialmente utilizada nas aulas iniciais: Olá Mundo! ou Hello World!.

Segue no abaixo o objetivo deste primeiro exercício:

Tags HTML

Faça uma página web que contenha as seguintes Tags HTML:

(O asterisco "*" nas Tags abaixo devem ser ignorados.)
Comentários
<*!-- -->
Parágrafo
<*p><*/p>
Destaque
<*h1><*/h1>, <*h2>, <*h3>... <*h6>
Contato
<*address>
Citações
<*blockquote>, <*q> e/ou <*cite>
Listas
<*ul>, <*ol> e <*li>
Listas de Definições
<*dl>, <*dt> e <*dd>
Link
<*a>

Exemplo de Lista Ordenada

  1. Farinha de trigo - 2/pct
  2. Leite UHT integral - 4/cx
  3. Açúcar Refinado - 1/pct
  4. Ovos - 12/unid
  5. Fermento em pó - 1

Exemplo de Lista Não Ordenada

  • Farinha de trigo - 2/pct
  • Leite UHT integral - 4/cx
  • Açúcar Refinado - 1/pct
  • Ovos - 12/unid
  • Fermento em pó - 1
