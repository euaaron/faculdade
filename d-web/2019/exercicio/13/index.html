Exercício 13

Faça um campo "CPF" que seja dinâmicamente formatado.

Passos:

  1. Verificar se o valor inserido já está no formato correto.
  2. Caso contrário, a cada valor que for inserido:
    1. Fazer uma cópia dos valores;
    2. Remover caracteres não numéricos;
    3. Retornar caracteres formatados.
